

Matlab声纹锁 使用教程

1.运行SpeakerUI

2.设置声纹密码：设置声纹密码需要训练三次

  （1）第1次：点选“重新设置密码”，点击“训练”，此时请对着电脑麦克风说一个词语，例如“芝麻开门”，等待5秒钟，提示第1次训练完成；

  （2）第2次：取消选择“重新设置密码”，点击“训练”，此时请对着电脑麦克风说同一个词语，例如“芝麻开门”，等待5秒钟，提示第2次训练完成；

  （3）第3次：取消选择“重新设置密码”，点击“训练”，此时请对着电脑麦克风说同一个词语，例如“芝麻开门”，等待5秒钟，提示第3次训练完成；

   (4）此时获取的三次训练音频样本，即为你相应的声纹密码。

3.密码测试：点击“测试”，对着麦克风说话。
  
  当说话人（同一个人）和说话内容（例如还是“芝麻开门”）和上述训练音频样本皆一致时，显示“密码正确”，否则显示“密码错误”或“再测一次”。
